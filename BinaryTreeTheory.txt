->  Binary Tree: Hierarchial Data Structure
    - At max 2 children
    - Levels and Subtree
    
    -> Pre-order Traversal : O(n)
        - Root
        - Left Subtree
        - Right Subtree
    
    -> In-order Traversal : O(n)
        - Left Subtree
        - Root
        - Right Subtree

    -> Post-order Traversal : O(n)
        - Left Subtree
        - Right Subtree
        - Root
